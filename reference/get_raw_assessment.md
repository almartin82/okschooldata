# Download raw assessment data from OSDE

Downloads OSTP assessment data from the Oklahoma State Department of
Education's public data files.

## Usage

``` r
get_raw_assessment(end_year)
```

## Arguments

- end_year:

  School year end (2017-2019, 2022-2025; no 2020-2021 due to COVID)

## Value

Data frame with raw assessment data
