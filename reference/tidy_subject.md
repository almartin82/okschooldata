# Tidy a single subject's data

Tidy a single subject's data

## Usage

``` r
tidy_subject(df, id_cols, prefix, subject_name)
```

## Arguments

- df:

  Wide-format data frame

- id_cols:

  Vector of identifier column names

- prefix:

  Column prefix (e.g., "ela", "math", "science")

- subject_name:

  Human-readable subject name

## Value

Tidy data frame for one subject
